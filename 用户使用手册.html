一、软件简介

本软件是一个基于 Python（Streamlit 框架）开发的气象数据交互分析平台，面向气象、环境、农业等领域的科研与业务场景。用户导入气象观测数据后，软件自动完成数据可视化、国家预警标准检测、智能应对建议生成，并支持导出分析报告。

📥

多源数据导入

支持 CSV、Excel、手动录入、API 自动获取四种方式

📈

专业可视化

综合看板、风向玫瑰图、散点矩阵、多站点对比

🚨

国家预警检测

8 类气象灾害预警，严格遵循中国气象局第16号令

💡

智能建议生成

公众出行建议 + 农业生产建议自动生成

二、安装与启动

1打开终端

按 Win + R，输入 cmd，回车打开命令提示符；或使用 Anaconda Prompt。

2进入项目目录

cd "C:\Users\ASUS\OneDrive\Desktop\24024115 郑昊\气象编程\weather_app"

3安装依赖（仅首次需要）

pip install -r requirements.txt
依赖包含：streamlit、pandas、numpy、plotly、matplotlib、openpyxl、python-docx、requests。若安装缓慢，可加 -i https://pypi.tuna.tsinghua.edu.cn/simple 使用清华镜像。

4启动软件

streamlit run app.py

启动后，终端会显示本地访问地址（默认为 http://localhost:8501），用浏览器打开即可使用。

⚠️ 注意：不要把 pip install 或 streamlit run 写进 .py 文件用 Spyder 运行——这些是终端命令，不是 Python 代码。Streamlit 是一个独立服务器进程，应通过外部终端启动。

四、Tab1 数据导入

这是使用软件的第一步。数据成功导入后，页面底部会显示"当前数据: N 条记录"。

4.1 文件导入（CSV / Excel）

  1. 在"📂 文件导入"子标签下，点击"上传 CSV 文件"或"上传 Excel 文件"
  2. 选择你的数据文件。软件会自动：
    • 识别列名：支持"气温/温度/temp"、"气压/pres"等多种中英文别名
    • 解析时间：自动识别 2026-07-01 14:00 等多种时间格式
    • 标记字段：展开"字段识别详情"可查看已识别/未识别的字段

4.2 手动录入

展开"添加观测记录"表单，逐条填写时间、气温、风速等，点击"➕ 添加记录"。适合少量补录。

4.3 API 自动获取

在"🌐 API 获取"子标签下：

  • 填入纬度、经度（默认北京燕郊 39.94, 116.85）
  • 选择日期范围
  • 点击"🔍 获取数据"——自动从 Open-Meteo（ERA5 再分析）下载逐时数据

💡 不确定数据格式？点击"📥 模板下载"获取 气象数据模板.csv，照着填即可。模板包含 timestamp（时间）、temperature（气温）、pressure（气压）等 11 个标准字段。

五、Tab2 数据质控

导入数据后，此页自动执行质量检查：

检查项说明
范围校验气温 -50~55℃、气压 500~1100hPa、湿度 0~100%、风速 0~75m/s
时间一致性检测相邻时次突跳（如 1h 降温≥8℃ 标记可疑）
完整性检查统计各字段缺失率、连续缺失段
质量评分综合计算百分制评分，辅助判断数据可用性

页面会以表格列出所有异常记录（红色标记），便于你清洗数据。

七、Tab4 智能分析与建议（核心功能）

本页是软件的核心价值所在，自动完成以下分析：

7.1 预警信号检测

严格依据《气象灾害预警信号发布与传播办法》（中国气象局第16号令）检测 8 类预警：

预警类型图标检测逻辑（节选）
高温黄 橙 红连续3天≥35℃ / 24h内≥37℃ / ≥40℃
寒潮蓝 黄 橙 红48h/24h 降温幅度 + 最低气温
大风蓝 黄 橙 红平均风力 6/8/10/12 级（10.8/17.2/24.5/32.7 m/s）
大雾黄 橙 红能见度 <500m / <200m / <50m
暴雨蓝 黄 橙 红12h/6h/3h 降雨量 ≥50mm / 3h≥100mm
霜冻蓝 黄 橙地温 ≤0℃ / ≤-3℃ / ≤-5℃
雷电黄 橙 红天气码含雷暴（WMO 95-99）
霾黄 橙能见度 <3000m / <2000m

7.2 多要素耦合分析

除单要素阈值外，还检测以下组合风险：

  • 热应激：高温（≥35℃）+ 高湿（≥60%）→ 计算体感热指数
  • 降水可能性：气压 6h 骤降（≥3hPa）+ 高湿（≥70%）
  • 风寒效应：低温（≤0℃）+ 大风（≥10.8m/s）

7.3 智能建议生成

页面下方自动生成两类建议（基于国家预警防御指南）：

  • 🚶 公众出行建议：如"高温橙色：尽量避免高温时段户外活动"
  • 🌾 农业生产建议：如"大风黄色：停止喷药和灌溉，加固设施农业骨架"
图 3：Tab4 预警卡片与建议生成效果

八、Tab5 气候态参照

输入经纬度后，软件拉取该地点近 5 年同期气候平均（ERA5），与你的数据对比：

  • 距平分析：标注"当前气温较同期偏高 X℃"
  • 异常判断：比单纯阈值更有意义——35℃ 在盛夏可能正常，在隆冬则是极端异常

九、Tab6 报文解码

面向专业气象场景。将标准气象报文粘贴入输入框，自动解析为结构化数据：

  • METAR：如 METAR ZBAA 010800Z 18006KT 5000 BR BKN040 25/18 Q1015
  • SYNOP：如 AAXX 01084 54511 02999 02501 ...

解析结果包含站点、时间、风、温湿、气压、能见度、云量等，可直接用于后续分析。

十一、侧边栏：自定义预警阈值

左侧栏的"⚙️ 自定义预警阈值"面板允许你调整各级阈值，以覆盖国家标准：

  • 高温预警：黄色/橙色/红色 触发温度
  • 大风预警：蓝/黄/橙/红 触发风速
  • 大雾预警：黄/橙/红 触发能见度

修改后点击"✓ 应用自定义阈值"即可生效。留空则使用国家预警标准。

⚠️ 适用场景：不同行业可能需要更严格的标准（如电力行业高温预警可能降至 38℃），自定义阈值用于此目的。

十二、常见问题（FAQ）

Q1：启动后浏览器打不开？

确认终端显示的地址（如 http://localhost:8501），手动复制到浏览器地址栏。若端口被占用，用 streamlit run app.py --server.port 8502 换端口。

Q2：列名识别不出来？

软件支持常见中英文别名。若你的列名特殊，请参考"📥 模板下载"中的标准字段名重命名，或在 Tab1 手动录入。

Q3：预警没有触发？

检查：① 数据量是否足够（高温需≥24条、寒潮需≥48条逐时数据）；② 字段名是否正确识别；③ 数据是否确实达到阈值。可在 Tab2 查看数据质量。

Q4：API 获取失败？

确认网络连通，且日期范围不超过 Open-Meteo 的免费区间（历史数据约 1940 年至今）。

Q5：Spyder 里运行 app.py 报错？

Streamlit 应通过外部终端（cmd / Anaconda Prompt）启动，不要在 Spyder 内用 %runfile 运行。Spyder 仅用于编辑代码。
